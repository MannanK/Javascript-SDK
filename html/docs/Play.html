Class: Play

Play

This class holds the play data returned from the feed servers. A typical play object looks like:

var play = {
  id: '123',
  station: {
    id: '456',
    name: 'foobar'
  },
  audio_file: [[ audio_file instance ]],
  liked: true
}

new Play()
